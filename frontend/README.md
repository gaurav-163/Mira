## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Backend Required

Make sure the FastAPI backend is running on port 8000:

```bash
cd ..
uv run python api.py
```

## Building for Production

```bash
npm run build
npm start
```
